# p2p-chat

## Structure

```
/p2p-chat
  /apps
    /web          # Next.js frontend
    /server       # Signaling server
  /packages
    /core
    /crypto
    /sockets
    ...
```

> Each `apps/*` directory should contain its own `package.json` file with at least the dev, build and start scripts defined
